## プロジェクト概要

ブラウザで遊べるかわいい鮫の癒しゲーム。3匹の性格豊かな小鮫に餌をあげたり、触れ合ったりして楽しむインタラクティブなウェブアプリケーション。

## 主要機能

### 1. 3匹の性格豊かな鮫

#### おっちょこちょい（ピンク色）
- ランダムに方向転換してしまう
- 餌を食べるのが苦手（判定範囲が小さい）
- 浮かぶ餌を優先的に狙う
- 速度: 1.2

#### よくばり（オレンジ色）
- 最も速く動く
- 全ての餌を追いかける
- 探知範囲が広い（500px）
- 速度: 2.5

#### おくびょう（薄青色）
- 他の鮫が近づくと逃げる
- ゆっくり沈む餌を優先的に狙う
- 餌に近づくのを躊躇する
- 速度: 1.5

### 2. 3種類の餌

- **通常の餌（オレンジ色）**: 通常の速度で沈む
- **ゆっくり沈む餌（青緑色）**: ゆっくり沈む、臆病な鮫が好む
- **浮かぶ餌（クリーム色）**: 水面で浮遊する、おっちょこちょいが好む

### 3. 満腹システム

- 30秒以内に20個の餌を食べると満腹状態に
- 満腹時: 1.5倍に太って動けなくなる
- 60秒後に自動回復
- 満腹中は餌が食べられない

### 4. その他の機能

- リセットボタン
- 満腹度システム
- 餌をあげた回数カウント
- レスポンシブ対応（スマホ・タブレット対応）
- タッチイベント対応

## 遊び方

1. ブラウザで `index.html` を開く
2. 画面をクリック/タップで餌をあげる
3. 鮫をクリック/タップして触れ合う
4. リセットボタンでゲームを初期化
